# Number-Guessing-Game
The Number Guessing Game is a simple interactive web game built using HTML, CSS, and JavaScript. The game generates a random number between 1 and 100, and the player has 10 attempts to guess the correct number. After each guess, the game provides feedback and keeps track of previous guesses and remaining attempts.
